# ADS MULTI-DB CONN
## RUN Locally
1. `git clone https://github.com/FaisalTmz007/simple-auth-ads-task.git`
2. `cd simple-auth-ads-task`
3. `npm install`
4. `cp .env.example .env`
5. `npm run migrate:db`
6. `npm run seed:db`
